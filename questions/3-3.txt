1. 给定几个变量 N, start，aim， k：
   N:           代表有从 1 到 N 个格子
   start：      代表初始位置
   aim：        代表目标位置
   k：          代表可以走 K 步
问：从 start 处走 k 步能到 aim 的方法数有多少？

例子：N = 4， start = 2， aim = 2，k = 2：
      
      1      2      3      4
           start
            aim

      第一种方法：2   ——>  1  ——>  2
      第二种方法：2   ——>  3  ——>  2 
      所以有 2 种方法。
